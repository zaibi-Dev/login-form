# Login Form

A simple login form built using HTML, CSS, and JavaScript.

## Features

- Email validation
- Password validation
- Error messages
- Login success message

## Technologies Used

- HTML
- CSS
- JavaScript
- react

## How It Works

- User enters email and password
- JavaScript validates input
- Errors shown if input is invalid
- Success message shown on valid login
